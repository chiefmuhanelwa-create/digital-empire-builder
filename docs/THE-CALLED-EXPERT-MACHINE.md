## 5 · THE CALLED EXPERT HANDLING SYSTEM (onboarding → result, automated)
*A unique, smooth path so every Called Expert gets help until they win. ~90% systemised.*

1. **Attract** — ICP1 content + ads → landing page (auto).
2. **Capture** — free Offer-Builder webapp / lead magnet → email + tag (auto).
3. **Diagnose** — Tally application (3Cs + 7-Stage self-assessment) → auto-scored (auto).
4. **Qualify** — qualified → Calendly discovery call; unqualified → nurture sequence (auto routing).
5. **Close** — 12-Step pipeline script (live, but scripted/SOP'd).
6. **Onboard** — Paystack paid → CHKPLT access + welcome sequence + first webapp unlocked (auto).
7. **Deliver** — 7-Stage Transformation modules (video) + interactive apps + weekly check-in automation.
8. **Result & Proof** — capture their win → testimonial → story bank → fuel for next cohort (the loop).
9. **Ascend** — graduate → done-for-you / affiliate / community contributor (3Cs Contribute).

**Founder touches only steps 5 & 7-video.** The rest is machines.
